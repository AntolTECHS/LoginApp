# LoginApp

LoginApp is a simple Android application that demonstrates user authentication with SQLite. Users can create an account, login, and change their password.

## Features

* User registration
* User login
* Change password functionality
* Local storage using SQLite

## Installation

1. Clone the repository:

   ```bash
   git clone git@github.com:AntolTECHS/LoginApp.git
   ```
2. Open the project in **Android Studio**.
3. Build and run the app on an emulator or physical device.

## Usage

1. Open the app.
2. Create a new account.
3. Login using the registered credentials.
4. Change your password if needed.

## Requirements

* Android Studio Flamingo or later
* Android SDK 36+
* Kotlin/Java support
